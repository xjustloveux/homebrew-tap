# Homebrew Tap

Third-party [Homebrew](https://brew.sh/) formulae provided by [xjustloveux](https://github.com/xjustloveux).

## Usage

```bash
brew tap xjustloveux/tap
brew install <formula>
```

Or install directly:

```bash
brew install xjustloveux/tap/<formula>
```

## Available Formulae

| Formula | Description | Install |
|---------|-------------|---------|
| [aspose-mcp-server](https://github.com/xjustloveux/aspose-mcp-server) | MCP server for office document processing powered by Aspose.Total | `brew install xjustloveux/tap/aspose-mcp-server` |

## Common Commands

```bash
# Upgrade a specific formula
brew upgrade <formula>

# Upgrade all formulae from this tap
brew upgrade xjustloveux/tap

# Uninstall
brew uninstall <formula>

# Remove this tap entirely
brew untap xjustloveux/tap
```
